## Build Process
1. Copy **src** folder to your ROS2 Workspace Folder
2. Run ```colcon build``` on your terminal

## Run Acquisition/Perception
From your ROS2 Workspace Folder:
```
. install/setup.bash
ros2 launch Perception perception.launch.py
```
